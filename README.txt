== Description ==

 This plugin adds custom fields to Ultimate Membership Pro membership forms and validates the number of webinars from Webinarpress a user can create based on their membership, in addition to hiding the admin bar and redirecting to the "Tutor" user after login

== Installation ==

1. Upload `ump-wp-limiter` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
